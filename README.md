
# 🚚 Supply Chain Logistics Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard designed to monitor and analyze supply chain operations. It provides insights into shipment performance, carrier efficiency, delivery delays, route events, and operational KPIs to support data-driven decision-making.

---

## 🎯 Objectives

- Monitor shipment performance
- Analyze delivery delays
- Evaluate carrier performance
- Track route events
- Improve logistics efficiency

---

## 🛠 Tools & Technologies

- Power BI
- DAX
- Power Query
- Microsoft Excel

---

### 📌 Executive Overview
- Total Shipments

### 📌 Carrier Performance

### 📌 Fuel vs Distance Analysis
- Distance Covered

### 📌 Route Event Monitoring
- Route Events
- Delay Hours
- Event Type
- Alert Status
- Anomaly Flag

---

## 📈 Key Insights

- Identified carriers with the highest delays.
- Compared transport modes based on shipment performance.
- Tracked operational events affecting deliveries.
- Monitored logistics KPIs for better decision-making.
